Descrição do Produto

Sistema completo de declarações românticas desenvolvido em Flask que permite aos usuários criar declarações personalizadas com fotos para seus entes queridos. Uma plataforma única para expressar amor de forma criativa e memorável.

Os usuários podem se cadastrar, fazer login, criar declarações personalizadas com nomes e fotos dos destinatários e compartilhá-las por meio de URLs exclusivas. O sistema inclui autenticação de usuário, gerenciamento de upload de arquivos e um painel completo para administrar as declarações.

Funcionalidades Principais:

  • Sistema de Cadastro e Login de Usuários
  • Autenticação Segura com Flask-Login
  • Criação de Declarações Personalizadas
  • Upload de Fotos com Pré-visualização
  • URLs Exclusivas para Compartilhamento
  • Painel de Controle com Estatísticas
  • Gerenciamento de Declarações
  • Processamento de Imagens com Pillow
  • Validação de Tipo de Arquivo
  • Limite de Tamanho de Upload (16 MB)
  • Proteção CSRF em Todos os Formulários
  • Senhas Criptografadas com Werkzeug
  • Página 404 Personalizada
  • Design Responsivo com Bootstrap 5
  • Tema Romântico Personalizado

Tecnologias Utilizadas:

  • Backend: Flask 3.1.2 com Roteamento Modular
  • Autenticação: Flask-Login para Gerenciamento de Sessão
  • Segurança: Flask-WTF para Proteção CSRF
  • Banco de Dados: SQLite com Esquema Simples
  • Frontend: Templates Jinja2 + Bootstrap 5
  • Processamento de Imagens: Pillow
  • Ícones: Font Awesome 6.4.0
  • CSS: Tema Romântico Personalizado
