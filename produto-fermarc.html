Descrição do Produto

Plataforma completa de e-commerce profissional desenvolvida em Python Flask com SQLite3, HTML5, CSS3 e JavaScript. Sistema robusto ideal para lojas de robótica, eletrônica e tecnologia, com todas as funcionalidades necessárias para uma loja virtual de sucesso.

Sistema profissional com autenticação segura, gerenciamento completo de produtos, carrinho de compras avançado, sistema de pedidos, cupons de desconto, lista de desejos, avaliações de produtos e painel administrativo completo. Pronto para deploy em produção.

O que está incluído:

  • Sistema de Usuários com CPF e Telefone
  • Autenticação Segura (Flask-Login)
  • Catálogo de Produtos com Paginação
  • Busca e Filtros Avançados
  • Carrinho de Compras Completo
  • Sistema de Cupons de Desconto
  • Lista de Desejos (Wishlist)
  • Sistema de Avaliações com Estrelas
  • Controle de Estoque Automático
  • Cálculo de Parcelamento
  • Desconto PIX (5%)
  • Gerenciamento de Pedidos Completo
  • Painel Admin com Dashboard
  • CRUD de Produtos e Categorias
  • Gerenciamento de Slides/Carrossel
  • Banco de Dados SQLite3
  • Pronto para Deploy (Gunicorn)
  • Design Responsivo Mobile-First
