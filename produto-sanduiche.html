Descrição do Produto

Plataforma de e-commerce completa desenvolvida em Python Flask, projetada especificamente para restaurantes, lanchonetes e estabelecimentos alimentícios. Uma solução profissional e escalável que oferece tudo que você precisa para vender online.

Sistema robusto com controle administrativo completo, chatbot com IA para atendimento automatizado, sistema de pedidos por QR Code para mesas, rastreamento em tempo real e integração com gateway de pagamento. Ideal para negócios que buscam modernização e eficiência operacional.

Funcionalidades Principais:

  • Catálogo de Produtos Dinâmico com Paginação e Busca
  • Carrinho de Compras Completo com Cálculo de Parcelas
  • Sistema de Pedidos com Rastreamento em Tempo Real
  • Painel Administrativo com Dashboard e Estatísticas
  • Gestão de Usuários com Diferentes Níveis de Permissão
  • Chatbot com IA para Atendimento Automatizado
  • Sistema de Pedidos por QR Code para Mesas
  • Integração com Mercado Pago (Gateway de Pagamento)
  • Controle de Estoque Automático
  • Sistema de Cupons e Descontos
  • Avaliações e Comentários de Produtos
  • Lista de Desejos

Recursos Avançados:

  • Cadastro Simplificado (Nome + Telefone)
  • Login via Número de Telefone
  • Autenticação por PIN de 4 Dígitos para Pedidos
  • Notificações em Tempo Real via Socket.IO
  • Controle de Área de Entrega por CEP e Raio
  • Sliders de Categorias na Página Inicial
  • Seção de Pedidos Recentes e Mais Vendidos
  • Exportação de Dados para CSV
  • Design 100% Responsivo e Mobile-First
