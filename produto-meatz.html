Descrição do Produto

Sistema completo de gerenciamento para hamburguerias, desenvolvido com as tecnologias mais modernas do mercado: Vite, Tailwind CSS e JavaScript puro. Frontend totalmente separado do backend, garantindo máxima flexibilidade e performance.
